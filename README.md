# Bigdata

Repositorio para almacenar y versionar el material de clases de la especialización en la Universidad CESMAG.

Aquí se guardan presentaciones, apuntes, guías, prácticas, código de ejemplo, datasets (cuando esté permitido) y otros recursos docentes.

Estructura recomendada:
- `asignatura/` — carpeta por asignatura.
- `asignatura/seccion/` — materiales por sesión o tema.

Uso:
- Clona el repositorio y navega a la carpeta de la asignatura que necesites.
- Respeta las indicaciones de los profesores sobre licencias y uso de datos.

Contribuciones:
- Profesores y estudiantes autorizados pueden agregar o actualizar material mediante commits y pull requests.

Contacto:
- Coordinación de la especialización — revisa la información dentro de cada asignatura.

Gracias por usar este repositorio para centralizar y preservar el material de la especialización.
